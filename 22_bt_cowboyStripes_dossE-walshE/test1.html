    Jumbotron

    Unlike in normal CSS, using Bootstrap we can create the "jumbotron" class.

    We used this feature to make that gray box at the top of the page. You can include the title here and whatever other stuff you want.

    Making Columns

    Bootstrap facilitates making columns that move when you resize the window. To do this you use the classes 'div' and 'col-sm-4'.

    Check out how we made these columns in Our References.

    Dropdown Menus

    Bootstrap allows you to make dropdown menus that remain fixed in place when you resize the window.

    Use "dropdown-menu" and "dropdown item" to indicate where the menu begins and ends. Check out Our References!
